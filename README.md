# Simple Contact Form PHP MySQL

Project ini adalah tugas Pemrograman Web untuk membuat form kontak sederhana menggunakan PHP dan MySQL.

## Fitur
- Form pengisian data kontak
- Menyimpan data ke database MySQL
- Menampilkan data kontak dalam bentuk tabel

## Teknologi yang Digunakan
- PHP
- MySQL
- HTML
- CSS (sederhana)

## Cara Menjalankan

1. Letakkan folder project ini di `htdocs` (XAMPP) atau `www` (Laragon).
2. Buat database bernama `kontak`.
3. Import tabel `kontak` (atau buat manual).
4. Buka browser, lalu akses:

http://localhost/simple-contact-form-php-mysql/kontak.html
